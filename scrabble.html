Scrabble solver

School project in 3rd year. Made with Java. An initial grid of scrabble, with all the special cases and the letters already placed, is given in a file, and another file contains the hand of the player. Using a specific algorithm the letters are placed optimally. It displays the result in the console and modifies the files accordingly.
